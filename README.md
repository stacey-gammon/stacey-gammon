## Build & deploy locally

### :wq
Install dependencies

```
yarn
```

ensure you have gatsby cli downloaded as well.

```
npm install -g gatsby-cli
```

### Build and run

```
yarn develop
```

### View

http://localhost:8000/

### Clean

```
gatsby clean
```
